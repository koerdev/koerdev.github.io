## Instalación y ejecución

Clona el repositorio:

```bash
git clone https://github.com/koerdev/koerdev.github.io.git
```

Instala dependencias:

```bash
npm install
```

Ejecuta en desarrollo:

```bash
npm run dev
```

Genera build:

```bash
npm run build
```
